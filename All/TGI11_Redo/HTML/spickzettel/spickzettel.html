Spickzettel für HTML

Tags:

<p>: Absatz
<h1>: Überschrift
<img>: Bild einfügen Attribut src="bilddatei": URL der Bilddatei (im Format jpg, gif, png) Attribut width="Angabe": Breite in Pixel (auch height für die Höhe) Attribut alt="Bildbeschreibung": kurze Beschreibung des Bildes
<hr>: waagrechte Linie

Sonderzeichen

Ä: &Auml;
Ö: &Ouml;
Ü: &Uuml;
ä: &auml;
ö: &ouml;
ü: &uuml;
ß: &szlig;
€: &euro;
„: &bdquo; (<!— linkes deutsches Anführungszeichen —>)
“: &ldquo; (<!— rechtes deutsches Anführungszeichen —>)
